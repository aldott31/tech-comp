# Albanian Tech Map

Interactive map of IT and digital companies in Albania, powered by automated scraping from the Albanian Business Registry (QKB).

## Versions

This repository contains modules for different Odoo versions:

- **[Odoo 17 Version](o17/albanian_tech_map/)** - For Odoo 17 installations
- **[Odoo 18 Version](o18/albanian_tech_map/)** - For Odoo 18 installations

Each version has its own README with detailed installation and usage instructions.

## Features

- Automated scraping of tech companies from Albanian Business Registry
- Interactive Leaflet.js map visualization
- Public web interface for viewing companies
- Automated data updates via scheduled cron jobs
- Selenium-based web scraping with headless Chrome
